1. Pig can be run in batch mode using __________ .
a) Pig shell command
b) Pig scripts
c) Pig options
d) All of the mentioned
Ans. b.Pig scripts

2. Which of the following is the default mode?
a) Mapreduce
b) Tez
c) Local
d) All of the mentioned
Ans. a)Mapreduce

3. Which of the following command can be used for debugging?
a) exec
b) execute
c) error
d) throw
Ans. a) exec

4. Which of the following function is used to read data in PIG?
a) WRITE
b) READ
c) LOAD
d) None of the mentioned
Ans. c)LOAD

5. Which of the following will run pig in local mode?
a) $ pig -x local …
b) $ pig -x tez_local …
c) $ pig …
d) None of the mentioned
Ans. a)$ pig-x local....

6. PigUnit runs in Pig’s _______ mode by default.
a) local
b) tez
c) mapreduce
d) None of the mentioned
Ans. c) mapreduce

7. Point out the wrong statement:
a) Pig can invoke code in language like Java Only.
b) Pig enables data workers to write complex data transformations without knowing Java.
c) Pig’s simple SQL-like scripting language is called Pig Latin, and appeals to developers already familiar with scripting languages and SQL.
d) Pig is complete, so you can do all required data manipulations in Apache Hadoop with Pig.
Ans. a)Pig can invoke code in language like Java Only.

8. Pig Latin is _______ and fits very naturally in the pipeline paradigm while SQL is instead declarative.
a) functional
b) procedural
